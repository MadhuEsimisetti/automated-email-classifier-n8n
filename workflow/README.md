# Automated Email Classifier using n8n

This project automates email classification and calendar scheduling using n8n, Gmail API, Google Gemini, and Google Calendar.

## Features
- Automatically classifies incoming emails
- Applies Gmail labels based on category
- Creates Google Calendar events for meeting emails

## Technologies Used
- n8n
- Gmail API
- Google Calendar API
- Google Gemini API

## How it Works
1. Gmail Trigger receives emails
2. AI Agent classifies the email
3. Labels are applied in Gmail
4. Meetings are added to Google Calendar
